Makes your site content visible only for several users who are in your trusted list.

== Description ==

Just add usernames of those who are allowed to read your site, then no other person will be able to view it.

== Installation ==

1. Unzip the plugin archive to the plugins directory.
2. Activate the plugin, then the Trusted Users link appears in the Users submenu.

== Frequently Asked Questions ==

= How to use the plugin? =

On the Trusted Users options page just list usernames of users who is allowed to view your site. Optionally, add a message to display on the login page (e.g. your contacts).
